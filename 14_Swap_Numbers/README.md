# Problem 14 - Swap Numbers

## 🚀 Problem Description
- Ask the user to enter **two numbers**.
- Swap the numbers.
- Display numbers **before and after swapping**.

**Example Input:**
Number 1: 10
Number 2: 20

**Example Output:**
Before Swap: Number1 = 10, Number2 = 20
After Swap: Number1 = 20, Number2 = 10

## 🛠 Solution Features
- **Layered OOP**: `Utils`, `InputHandler`, `NumberSwapper`, `ApplicationFlow`.
- **Template-based**: Works with `int`, `float`, `double`.
- **Validated input**: Ensures user enters valid numbers.
- Swap logic centralized in `NumberSwapper` class.

## 🚀 How to Run
1. Compile `14_Swap_Numbers.cpp`.
2. Run the executable.
3. Enter two numbers as prompted.
4. Observe numbers before and after swap.
