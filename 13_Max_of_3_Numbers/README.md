# 🚀 Problem 13: Max of 3 Numbers

This problem determines the **maximum of three numbers** entered by the user using a **layered OOP architecture**.

---

## 🛠 Architectural Overview
- **Utils**: Handles input stream clearing and validation utilities.
- **InputHandler**: Ensures validated numeric input.
- **MaxCalculator**: Encapsulates the business rules for finding max between 2 or 3 numbers.
- **ApplicationFlow**: Orchestrates reading input, processing, and displaying the result.

---

## 📌 Problem Description
- Ask the user to enter **three numbers**.
- Display the **largest number**.
- Use **OOP** to separate concerns and handle input validation.

**Example Input:**
10
25
20

**Example Output:**
Max Number = 25
